# Ayudantía 01 - Diseño de Software

En esta ayudantía revisamos y aplicamos los siguientes conceptos y herramientas:

- **Estructura de repositorio de un proyecto en HTML:** Organizar assets, carpeta de clases, y el uso de múltiples archivos `.html`.
- **Importación de recursos:** Cómo integrar fuentes (fonts), archivos CSS e imágenes en el proyecto.
- **Herramientas y páginas útiles:** Uso de [Google Fonts](https://fonts.google.com/) para tipografías y [Font Awesome](https://fontawesome.com/) para íconos.
- **Estructura del Body y Flexbox:** Estructuración del `body` usando `flex` y análisis de sus diferencias principales con `grid`.
- **Selectores CSS específicos:** Asignación de estilos a etiquetas específicas dentro de una clase (por ejemplo, aplicar estilos solo a etiquetas `h1` que estén dentro de una clase CSS particular).
- **Pseudo-clases:** Uso del estado `:hover` para interacciones.
- **Variables y funciones:** Uso de la función `calc()` y declaración de variables CSS en `:root`.
- **Efectos visuales:** Implementación de `box-shadow` para sombras.
- **Espaciado:** Uso de `gap` para manejar la separación entre elementos en flexbox/grid.
- **Animaciones/Transformaciones:** Uso de traslación en el eje Y (`transform: translateY()`) para lograr efectos de movimiento.
